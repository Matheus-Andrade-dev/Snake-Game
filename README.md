Jogo da Cobrinha (Snake Game)

Projeto de um Jogo da Cobrinha desenvolvido com HTML, CSS e JavaScript, utilizando a tag <canvas> para renderização do jogo.
Segui a base apresentada no vídeo do Manual do Dev, adicionando melhorias próprias, como a persistência do High Score usando LocalStorage, permitindo que a melhor pontuação seja mantida mesmo após fechar o navegador.
